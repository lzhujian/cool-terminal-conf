# cool-terminal-conf

Tmux配置文件和vim插件，用于打造一个高效易用的开发者终端环境

## 安装

直接运行 `install.sh` 脚本进行安装，该脚本将tmux配置文件拷贝到 ~/.tmux.conf，将 .vim* 拷贝到 ~ 目录

```shell
./install.sh
```
